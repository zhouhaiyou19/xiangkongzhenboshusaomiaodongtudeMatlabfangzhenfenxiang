## 内容简介

相控阵天线是由多个天线阵元组成的集合，每个阵元的辐射图在结构上与相邻天线的辐射图合成，形成主瓣的有效辐射图。主瓣在期望的方向上辐射能量，而波束设计的目标是在不需要的方向上形成零点和旁瓣。

本资源通过MATLAB仿真，展示了相控阵天线在不同扫描角度下的波束赋形过程，并生成了动态的波束扫描图。通过这些动图，用户可以直观地观察到相控阵天线在不同方向上的辐射特性，以及波束扫描的效果。
